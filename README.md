# 📦 全日物流 / 同興出庫單 - 自動化比對工具

這是一個專為物流人員設計的 Web App，旨在快速比對「全日物流貨運出貨單」與「同興出庫單」之間的資料差異，避免人工肉眼核對造成的失誤。

## ✨ 功能亮點
* **自動化聚合**：自動針對「客戶單號」進行 Group By 加總，處理拆單分列問題。
* **精準比對**：自動篩選「出庫」單據，比對數量(副)是否與出庫數量一致。
* **差異導出**：一鍵匯出 Excel 差異表，方便後續異常處理。
* **隱私安全**：所有比對邏輯均在瀏覽器本地執行，資料**不會**上傳到任何伺服器。
* **直觀介面**：採用類 Excel 網格介面，差異數據一目了然。

## 🛠 使用步驟
1.  **上傳全日物流單**：將 Excel 檔案拖入左側區塊（系統將讀取第一個 Sheet）。
2.  **上傳同興出庫單**：將 Excel 檔案拖入右側區塊。
3.  **開始比對**：點擊按鈕，系統將自動進行邏輯運算。
4.  **查看結果**：下方「比對結果」將僅顯示**數量不一致**的異常單據。
5.  **下載報告**：如有差異，可點擊「匯出差異 Excel」。

## 📊 比對邏輯說明
* **全日物流單**：
    * 篩選條件：C 欄（單據別）為「出庫」。
    * 計算邏輯：以 L 欄（客戶單號）為基準，加總 I 欄（出庫數量）。
* **同興出庫單**：
    * 計算邏輯：以 C 欄（單號）為基準，加總 L 欄（數量(副)）。
