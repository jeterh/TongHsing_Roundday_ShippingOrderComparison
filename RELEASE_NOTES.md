# Release Notes - v1.0.0 (Initial Release)

**發布日期：** 2024-05-22
**版本狀態：** Stable

這是「全日/同興比對工具」的第一個正式版本。此版本專注於建立核心的比對邏輯與直觀的用戶操作流程。

### 🆕 新增功能
* **Excel 解析引擎**：支援 .xlsx 與 .xls 格式讀取。
* **聚合運算模組**：支援多列相同單號的數量自動累加。
* **異常捕捉介面**：自動比對並列出「單號」、「原數量」、「目標數量」及「差異值」。
* **檔案檢查機制**：新增檔案未上傳時的彈窗錯誤提示。
* **Excel 導出**：支援將比對後的差異清單下載為 Excel 報表。

### 🛠 修復與優化
* 優化大數據量下的比對效能，減少 UI 暫停響應。
* 修正欄位索引，精準對應全日物流 C, I, L 欄與同興 C, L 欄。
* 加入響應式設計，支援平板與電腦螢幕操作。

### ⚠️ 已知限制
* 目前僅支援讀取 Excel 的第一個工作表 (Sheet)。
* Excel 格式需符合預設範本欄位順序（全日 L 欄為單號，同興 C 欄為單號）。
